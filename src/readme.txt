icotoprc
========

A tool for converting a Windows icon file (.ico) to a format useable 
by the PWLib application "pwrc"

To be able to compile this application you will need a recent copy 
of the pwlib source (www.openh323.org) and follow the instructions 
on that site to compile this library.

The open the icotoprc.dsw file in Visual Studio 6

Compile icotoprc in the way you would normally do this in Visual 
Studio

To run the application:
c:\....\>icotoprc -f test.ico

The output can be used in a prc file, see the files supplied by the 
OpenH323 project
